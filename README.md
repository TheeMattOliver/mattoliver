## 🚀 Run locally:

1.  **Install.**

    ```shell
    # install dependencies
    npm install
    ```

1.  **Start developing.**

    Navigate into the site’s directory and start it up.

    ```shell
    npm start
    ```

1.  **Open the source code and start editing!**

    Site will run at `http://localhost:8000`.
